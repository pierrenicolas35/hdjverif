Détermination de l’éligibilité à l’hospitalisation de jour
Akinator HDJ est une application web simple permettant de déterminer, via une succession de questions, si un patient est éligible à une facturation en hôpital de jour (HDJ) selon les critères définis par les autorités de santé françaises.

L’objectif est d’aider les professionnels (DIM, cadres de santé, médecins, facturation) à vérifier la conformité d’une prise en charge.
